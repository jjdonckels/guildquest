## Overview
This project is a mini-adventure game framework built for the GuildQuest world. It supports two  playable game modes where players interact with entities such as enemies, hazards, items, and NPCs on a grid-based board. 


## How to Run
1. Download gmae.jar from the root directory.
2. Make sure you have jdk 25 or newer.
3. Run the executable jar file. 
```bash
java -jar gmae.jar
```
